title: 解离（Dissociation）

**解离（Dissociation）** 指个体在知觉、意识或认知层面上，与外界环境或自身身体体验发生解离、断裂或去整合的状态。该现象常见于**解离剂**（如右美沙芬、美金刚等）作用下，其核心神经生物学机制主要涉及 **NMDA 受体拮抗**：相关物质与受体结合但不激活，抑制谷氨酸介导的兴奋性信号传递，使中枢神经系统的信息整合能力呈**剂量依赖性下降**，从而引发意识、感官与自我体验的系统性“脱耦”。

下文对常见的解离性体验类型作系统化概述。

---

## 认知脱节（Cognitive Dissociation）

认知脱节指个体对自我身份、思维流或高级认知过程产生疏离感，常见表现包括：

- 人格解体（Depersonalization）
- 现实解体（Derealization）
- 记忆抑制或片段化
- “自我消融”（Ego death）
- 整体认知功能受限

**主观特征**：

- 思维或记忆被感知为“非自我所有”
- 思维变得遥远、模糊或断续
- 行为呈现自动化或机械化
- 认知速度、连贯性与执行功能下降

该状态常与视觉或身体脱节并行出现，尤以**中等剂量**解离剂作用下最为典型。

---

## 似曾相识（Déjà vu）

似曾相识指个体在全新情境中产生异常而强烈的熟悉感，其特征包括：

- 非理性的熟悉体验
- 明知情境全新却无法消除熟悉感

在解离剂或致幻药物影响下，该体验常伴随：

- 对环境或自身动作的异常强化感知
- 药物效应引发的整体压倒性感受

部分个体还会出现嗅觉幻觉或现实解体体验，多见于**中等剂量**阶段。

---

## 解离高原（Dissociative Plateaus）

解离高原是一种经验分级模型，用于描述解离剂所致的**认知、身体与视觉脱节程度**，常应用于氯胺酮、PCP、右美沙芬等物质的体验分类。

| 高原级别   | 描述                                                                 |
| ---------- | -------------------------------------------------------------------- |
| Ⅰ 微妙解离 | 感官轻度迟钝，环境体验略显疏远                                       |
| Ⅱ 部分解离 | 感知距离拉远，出现复视或模糊；触觉麻木，运动协调受限                 |
| Ⅲ 完全解离 | 感官与认知基本脱离外界，进入“解离洞”，伴随灵魂出窍与内部幻觉         |
| Ⅳ 自我解离 | 长期记忆与自我表征暂时中断，出现明显“自我消融”                       |
| Ⅴ 脱离意识 | 感官与认知完全丧失，意识近似空白并伴随长时间失忆，常见于麻醉相关状态 |

---

## 身体脱节（Physical Dissociation）

身体脱节指个体对身体所有权、触觉或运动控制的疏离体验，常见表现包括：

- 触觉与疼痛感减弱
- 自主或“被动”运动感增强
- 身体形态或比例知觉改变
- 轻盈、漂浮或重力异常感
- 整体身体功能抑制

**主观特征**：

- 身体被体验为“非自我之物”
- 动作执行呈现自动或程序化
- 精细运动能力下降
- 触觉敏感性显著降低

该状态通常与认知及视觉脱节同时出现，常见于**中等剂量镇静性或解离性药物**作用下。

---

## 视觉脱节（Visual Dissociation）

视觉脱节指个体对外界视觉输入的整合与归属感下降。低级别表现包括：

- 视力范围或清晰度下降
- 复视或图像重叠
- 图案与物体识别能力减弱
- 视觉处理速度减慢

**主观体验**：

- 仿佛透过屏幕或滤镜观察世界
- 细节难以聚焦，整体画面遥远
- 产生“借用他人视角”的观感

在高级阶段，外界视觉输入可完全消失，并被内部视觉体验或幻觉所取代。

### K 洞（K-Hole）

K 洞是高强度视觉与认知脱节的典型形态，其特征包括：

- 外部视觉彻底消失，被“主观空间”取代
- 空间多呈巨大、黑暗、空旷结构，偶见缓慢移动的色块或几何形态
- 重力与身体定位感瓦解，伴随强烈灵魂出窍体验

### 结构（Structures）

在解离性主观空间中，个体常感知到复杂的“结构”形态：

- 三维形体的生成、旋转、缩放与变换
- 常见意象：柱体、隧道、建筑群、纪念碑、金字塔、分形结构
- 整体美学偏暗，呈现外星化或科幻特征

**复杂度分级**：

| 级别             | 描述                                                                       |
| ---------------- | -------------------------------------------------------------------------- |
| 2D 结构          | 扁平、深色，仅具存在感，无明确尺度                                         |
| 部分定义 3D 结构 | 半透明、边缘模糊，具基础光影，尺度达数百米                                 |
| 完全定义 3D 结构 | 边缘清晰、材质逼真，尺度可达数千英里，细节高度复杂                         |
| 结构化机器宇宙   | 整个宇宙被体验为无限复杂、动态演化的机械结构，同时具备视觉、认知与触觉沉浸 |

**结构过渡方式**：

- **形态变换**：结构连续形变
- **平移切换**：当前结构移出视野，新结构进入
- **远距穿越**：个体沿隐性轨道高速漂移，穿行于无限虚空中的结构体系
